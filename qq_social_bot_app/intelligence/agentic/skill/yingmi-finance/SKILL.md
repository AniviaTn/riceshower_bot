# 盈米金融工具集

本 Skill 加载盈米 MCP Server 提供的金融类工具，用于查询和分析基金、理财产品等金融数据。

## 能力范围

通过盈米 MCP 工具集，你可以：

- **基金查询**：按名称、代码、类型检索基金信息
- **净值与收益**：查询基金历史净值、区间收益率、业绩排名
- **持仓分析**：查看基金持仓明细、行业分布、重仓股
- **理财产品**：查询各类理财产品信息与对比
- **投资组合**：分析组合配置、风险收益特征

## 使用说明

当收到金融相关问题时，直接调用盈米 MCP 工具集中可用的工具进行查询。工具列表会在加载后自动注入到可用工具中，根据用户的具体需求选择合适的工具调用即可。
